# Development

Scripts and procedures for developing locally.

## Getting started

Install node modules

```
pnpm install
```

Start storybook

```
pnpm start
```

## Testing

Run Vitest tests

```
pnpm test
```

## Prettier

Run prettier on source files

```
pnpm prettier
```

## Publishing New Version to NPM

Use `np`

```bash
npx np
```
